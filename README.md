# Landing de Servicios de Desarrollo

Landing page profesional para servicios de desarrollo de software, creada con React, Vite y CSS puro.

## Características

- Diseño moderno y responsive
- Animaciones suaves
- Secciones completas: Hero, Servicios, Sobre Mí, Contacto
- CSS puro (sin frameworks)
- Optimizado para rendimiento

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El proyecto se ejecutará en `http://localhost:5173`

## Construcción

```bash
npm run build
```

## Estructura del Proyecto

```
src/
├── components/          # Componentes React
│   ├── Header.jsx
│   ├── Hero.jsx
│   ├── Services.jsx
│   ├── About.jsx
│   ├── Contact.jsx
│   └── Footer.jsx
├── styles/             # Estilos CSS
│   ├── index.css
│   ├── Header.css
│   ├── Hero.css
│   ├── Services.css
│   ├── About.css
│   ├── Contact.css
│   └── Footer.css
├── App.jsx
└── main.jsx
```

## Personalización

1. Actualiza los textos en los componentes según tus servicios
2. Modifica los colores en `src/styles/index.css` (variables CSS)
3. Agrega tu email en el componente Contact
4. Personaliza los servicios en `Services.jsx`
5. Actualiza la información en `About.jsx`

## Tecnologías

- React 18
- Vite 5
- CSS puro
